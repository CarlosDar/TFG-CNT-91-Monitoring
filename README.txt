INSTRUCCIONES DE EJECUCIÓN
=========================

Para ejecutar la aplicación, sigue estos pasos:

1. Abre una terminal o línea de comandos
2. Navega hasta la carpeta del proyecto
3. Ejecuta el siguiente comando:
   cd src/web && python app.py

La aplicación se iniciará y se abrirá automáticamente en tu navegador web predeterminado en la dirección:
http://127.0.0.1:5000

Nota: Asegúrate de tener Python instalado en tu sistema antes de ejecutar la aplicación.

NAVEGACIÓN POR LA APLICACIÓN WEB
===============================

La aplicación web está dividida en tres secciones principales, accesibles desde el menú lateral:

1. Inicio
   - Página de inicio con información general del dispositivo
   - Botón para la conexión del dispositivo
   - Botón para acceder a la configuración
   - Botón para acceder a las mediciones
   - Botón para acceder a la información CNT-91 

2. Configuración
   - Permite configurar los canales del dispositivo
   - Selección entre Canal A y Canal B
   - Botón para aplicar la configuración

3. Mediciones
   - Visualización de datos en tiempo real
   - Gráficos y valores medidos
   - Control de las mediciones

4. Información CNT-91
   - Descripción general del dispositivo
   - Características principales
   - Enlaces a documentación oficial
   - Manuales y recursos técnicos

En la parte superior de la aplicación encontrarás:
- Indicador de estado del dispositivo (Conectado/Desconectado)
- Botón para conectar/desconectar el dispositivo
