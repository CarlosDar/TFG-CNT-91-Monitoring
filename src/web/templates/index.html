Información del CNT-91

Descripción General

El CNT-91 es un contador/temporizador universal de alta precisión diseñado para mediciones de frecuencia, período, tiempo y otros parámetros relacionados.

Características Principales

  • Rango de frecuencia: DC a 300 MHz
  • Resolución de tiempo: hasta 100 ps
  • Múltiples canales de entrada
  • Interface GPIB para control remoto
  • Pantalla LED de alta visibilidad

Aplicaciones

  • Mediciones de frecuencia de alta precisión
  • Calibración de equipos
  • Investigación y desarrollo
  • Control de calidad
